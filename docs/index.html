hm: a Python framework for hydrological modelling¶

hm is a free and open source project for building environmental models in Python.

hm relies heavily on xarray for handling spatio-temporal datasets. The underlying data model is provided by the netCDF file format. Methods for timestepping are inspired by those included in PCRaster.
